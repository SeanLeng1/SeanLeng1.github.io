# Jixuan Leng Website

Personal website built with Jekyll. The current version keeps the Jekyll build pipeline and rebuilds the UI around a lighter homepage plus a gallery-style publications archive.

Editable content is organized as:

- `_data/home.yml`: homepage section content
- `_data/profile.yml`: avatar, email, social links, contact info
- `_data/navigation.yml`: top header brand mark and navigation labels
- `_data/music.yml`: music player button label and playlist configuration
- `_bibliography/papers.bib`: publications metadata and selected papers

`legacy/` is preserved as the old site and is intentionally left untouched.
